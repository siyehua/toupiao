# 吉尔吉斯高校大学生消费行为调查问卷

这是一个针对吉尔吉斯高校大学生的消费行为调查问卷，支持中文和俄语双语。

## 功能特点

- 响应式设计，适配移动端
- 中俄双语支持
- 完整的问卷内容
- 表单验证功能

## 本地开发

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm start
```

## 部署

使用 Vercel 进行部署：

1. 安装 Vercel CLI：
```bash
npm install -g vercel
```

2. 部署到 Vercel：
```bash
vercel
```

## 技术栈

- HTML5
- CSS3
- JavaScript
- Vercel 部署
